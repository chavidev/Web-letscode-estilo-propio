HTML

  • Etiquetas:
    • <a> Define un hipervínculo
    • <article> Define un artículo
    • <aside> Define el contenido lateral del contenedor de una página
    • <button> Define un botón clickeable
    • <figure> Especifica auto-contenido
  • Atributos:
    • alt Texto alternativo en caso de que la imagen no se pueda mostrar.
      • Uso en:
        • <applet>
        • <area>
        • <img>
        • <input>
    • autofocus El elemento debería recibir foco automáticamente después de haberse cargado la página.
      • Uso en:
        • <button>
        • <input>
        • <keygen>
        • <select>
        • <textarea>
    • charset Declara la codificación de caracteres de la página o del script.
      • Uso en:
        • <meta>
        • <script>
    • draggable Define si el elemento puede ser arrastrado.
      • Uso en:
        • Atributo Global

CSS

  • selectores de tipo:
    • todos los elementos que coinciden con el nombre del elemento especificado.
    • Ejemplo:
      • a
      • span
      • img
  • selectores de clase:
    • todos los elementos que tienen el atributo de class especificado.
    • Ejemplo:
      • .nombre-clase
  • selectores de ID:
    • selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.
    • Ejemplo:
      • #nombre_id
  • selector universal:
    • todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres.
    • Ejemplo:
      • * se aplicará a todos los elementos del documento.
  • selector de atributo:
    • selecciona elementos basándose en el valor de un determinado atributo.
    • Ejemplo:
      • [attr*=value]
      • Selecciona los elementos cuyo atributo attr tenga un valor que contenga value.
