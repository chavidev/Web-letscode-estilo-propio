HTML

    • <a> Define un hipervínculo
    • <article> Define un artículo
    • <aside> Define el contenido lateral del contenedor de una página
    • <button> Define un botón clickeable
    • <figure> Especifica auto-contenido
  • Atributos:
    • alt Texto alternativo en caso de que la imagen no se pueda mostrar.
      • Uso en:
        • <applet>
        • <area>
        • <img>
        • <input>
    • autofocus El elemento debería recibir foco automáticamente después de haberse cargado la página.
      • Uso en:
        • <button>
        • <input>
        • <keygen>
        • <select>
        • <textarea>
    • charset Declara la codificación de caracteres de la página o del script.
      • Uso en:
        • <meta>
        • <script>
    • draggable Define si el elemento puede ser arrastrado.
      • Uso en:
        • Atributo Global
